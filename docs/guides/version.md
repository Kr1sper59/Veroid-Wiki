# Смена версии
## Визард
Визард - вкладка для управления версиями и ядрами

### Установка
1. Выберите нужную вам категорию ядер.
2. Выберите конктерное ядро.
3. Выберите нужную вам версию игры.
4. Выберите доп действия при установке(по желанию).

## Ручная установка 
### Установка ядер для плагинов и прокси
1. Скачайте нужное вам ядро с официального сайта.
2. Загузите его на сервер, и переименуйте в server.jar(либо в имя файла, указанное во вкладке запуск).
3. Выберите нужную версию Java согласно таблице в конце статьи.

### Установка ядер для модов
1. Скачайте установщик вашего ядра.
2. Запустите его, и выберите установку сервера.
3. Удалите все файлы с сервера, и загрузите те, которые создал установщик.
4. Выберите нужную версию Java согласно таблице в конце статьи.

## Автоматическая установка
### Установка ядер для плагинов и прокси
1. Укажите нужную версию во вкладке **Запуск**.
2. Перейдите во вкладку **Настройки**, и нажмите Reinstall server(переустановить сервер)
3. Выберите нужную версию Java согласно таблице в конце статьи.

### Установка ядер для модов
1. Укажите нужную версию во вкладке **Запуск**.
Версию нужно указывать в окне ВЕРСИЯ FORGE. Например: `1.20.1-47.4.13`, где 1.20.1 - версия Minecraft, а 47.4.13 - версия Forge.
2. Перейдите во вкладку **Настройки**, и нажмите Reinstall server(переустановить сервер)
3. Выберите нужную версию Java согласно таблице в конце статьи.
