# Плагины
Плагин - дополнение для ванильных серверов, например Paper. Не путать с модами!

## Откуда скачивать плагины
Плагины для сервера лучше всего скачивать с проверенных источников, таких как Modrinth, SpigotMC, GitHub, Официальный сайт плагина(Если есть).

Плагины с непроверенных источников могут навредить вашему серверу, и лучше их не ставить.

## Как устанавливать плагины
Для начала, нужно скачать плагин под вашу версию и [ядро](/guides/kernel). Иначе есть большой шанс, что плагин просто не будет работать.

После того, как вы скачали плагин, его нужно загрузить на сервер в папку plugins. Это можно сделать через панель управления, или [SFTP](/guides/sftp).

Затем перезагрузите сервер.

**Готово, плагин установлен!**
