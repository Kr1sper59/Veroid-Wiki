# Как сделать переливающийся градиент в табе?

qq all, темка будет базироваться на анимации плагина TAB

Шаг 1. Зайти на сайт https://www.birdflop.com/resources/animtab/

Шаг 2. Выбрать подходящие цвета к вашему элементу Таба, будь то префикс или название вашего сервера

Шаг 3. Настроить саму анимацию и определить ей название (Я лично рекомендую параметр Gradient Length ставить в 2 раза больше чем символов в слове. То есть: 5=10, 9=18 и т.д.)

Шаг 4. Скопировать код который получился.

Шаг 5. Перейти на container/plugins/TAB/animations.yml, и вставить анимацию.

Все готово, у вас есть своя анимация!
Чтобы её использовать используйте плейсхолдер %animation:названиеанимки%. Удачи с созданием анимаций :)
