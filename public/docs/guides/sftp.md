# Подключение по SFTP

SFTP - протокол для защищенной передачи файлов.

## Как подключиться

### Данные для подключения

Данные для подключения можно найти в панели управления во вкладке **Настройки**.
Для подключения к SFTP используйте пароль от панели.

![Параметры подключения к SFTP](/assets/sftp-connection.png)

### 1. Termius

Скачать можно тут: [termius.com](https://www.termius.com/download/)

1. Создайте новый хост для подключения.
1. Заполните необходимые данные:
   - **IP or Hostname** - IP-адрес сервера без порта (например: **bullet.veroid.ru**).
   - **Label** - имя сервера (необязательно).
   - **SSH port** - порт для подключения к SFTP (например: **2002**).
   - **Username** - имя пользователя для подключения к SFTP.
   - **Password** - пароль от панели управления.
1. Нажмите кнопку SFTP слева сверху и выберите ранее добавленный хост.

### WinSCP

Скачать можно тут: [winscp.net](https://winscp.net/eng/download.php)

1. Создайте новое подключение.
1. Выберите протокол передачи **SFTP**.
1. Заполните данные для подключения:
   - **Имя хоста** - IP-адрес сервера без порта (например: **bullet.veroid.ru**).
   - **Порт** - порт для подключения к SFTP (например: **2002**).
   - **Имя пользователя** - имя пользователя для подключения к SFTP.
   - **Пароль** - пароль от панели управления.
1. Нажмите кнопку **Сохранить** для дальнейшего использования и выберите созданный сервер в панели слева.

> Если подключение не проходит, проверьте порт, логин и не содержит ли пароль лишних пробелов.
